TMIengine library

A Library by Corsini Lorenzo for the Processing programming environment.
Last update, 08/18/2017.

A little the incredible machine remake, based on Fisica physics engine and processing.

Download

Download TMIengine library version 1.0.0 (1) in .zip format.

Installation

Unzip and put the extracted TMIengine folder into the libraries folder of your Processing sketches. Reference and examples are included in the TMIengine folder.

Keywords. incredible machine game
